Add User Language and Search Language

 

No.

Name

Description

1

Add Skill Label

Fixed Label for Add Skill to indicate that the user can add their user skills on this screen.

2

Search Skill Label

Fixed Label for Search Skill to indicate that the user can search a skill they would like to add to their profile.

3

User Details Grid

A grid used to list all the skills in the system that aren’t already in the users profile

4

Close Button

Button that submits and closes the add skills screen.

5

Search Skill Textbox

Input of type text to enter in the skill the user wishes to search for.

6

Add Skill Button

Button that submits and add the corresponding skill to the users profile.
