Add Table

This functionality describes the process whereby the Systems Admin adds a table to the system for a user to book. The Systems Admin navigates to the table screen. The list of existing tables, slot and add table and disable date will be displayed. The Systems administrator clicks on the plus button to add a floor. The Systems admin will select the building name and enter a floor number and this new table will be added to the system.
