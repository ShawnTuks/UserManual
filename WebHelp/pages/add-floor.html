Add Floor  

This functionality describes the process whereby the Systems Admin adds a floor. The Systems Admin navigates to the company screen. The list of existing floors in the building will be displayed. The Systems administrator clicks on the plus button to add a floor. The Systems admin will select the building name and enter a floor number and this new floor will be added to the system
