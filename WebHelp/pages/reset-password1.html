Reset Password  

 

No.

Name

Description

1

Reset Password Label

Fixed Label that tells the user to Reset their Password

2

New Password Hint

Hover Hint Button that gives the user a hint on what to enter into the new password textbox, when hovered it says “Ensure your password has a minimum of 8 characters containing 1 upper case letter and 1 number.”

3

Confirm Password Hint

Hover Hint Button that gives the user a hint on what to enter into the new password textbox, when hovered it says, “Enter in the matching password as to the password you just entered.”

4

Close Button

Button that closes the pop-up screen and returns the user to their edit profile screen.

5

New Password Textbox

Input of type text to enter in the user’s valid new password

6

Confirm Password Textbox

Input of type text to enter in the user’s valid confirm password

7

Reset Button

Button that submits reset password input.
