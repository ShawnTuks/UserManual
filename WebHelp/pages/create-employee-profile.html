Create Employee Profile  

This functionality describes the process of an HR Manager creating an employee. The HR Manager will begin the process on the home screen and request to create a employee. The system will take the HR Manager to the create employee profile screen on which the HR Manager will be asked to enter the job position, salary, schedule, start-date, end-date , location, email  and roles. The HR manager will then save the employee to the database.
