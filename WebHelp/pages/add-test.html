Add Test  

This functionality describes the process whereby the Systems Admin adds a test. The Systems Admin navigates to the Job Card screen. The list of existing tests will be displayed. The Systems administrator clicks on the plus button to add a test. The Systems admin will add the test name, description, and URL this new test will be added to the system.
