Verify Account  

This functionality verifies the users account so that the user will be able to use the system. If a user has not verified their user account, the user will not have access to the system. A verification link will be sent to the users email address once their applicant or employee profile has first been created. The user will be required to go into their email address which they will then find an email from the company which will contain a link. The user must select this link which will transfer the user to the verification screen. By the user selecting this link, the system will verify the users account. The user can then go onto the login page and login to the system.
