Add Team Member  

This functionality describes the process whereby the Systems Admin adds a team member to a team. The Systems Admin navigates to the teams’ screen. The list of existing teams will be displayed. The Systems administrator will select a team and clicks on the plus button to add a team member. The Systems admin will search for a user to add to the team. The new member is added to the team.
