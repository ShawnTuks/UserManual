Add Operation Authorisation  

This functionality describes the process whereby the Systems Admin adds an operation authorization. The Systems Admin navigates to the Operation and Authorizations screen. The list of existing operation authorizations and will be displayed. The Systems administrator adds a role affected, target, operation, and database table. This will then add the operation authorisation to the system.
