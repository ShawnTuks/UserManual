Disable Date

This functionality describes the process whereby a systems admin disables a date from the system which prevents any booking from being able to book that day. The systems admin will select the tables screen, which the system will correspondingly display. The systems admin will navigate to the dates section and uncheck the date which they wish to disable. The system will save this transaction and remove any bookings from that date which have been previously booked.
