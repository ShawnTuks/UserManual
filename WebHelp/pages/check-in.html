Check-In  

This functionality describes the process of an employee checking into a table they have previously booked. This functionality is done through the mobile application. The user will select to check in and will be required to scan the barcode on the table they wish to check in to. Once the user has scanned the table, the system will confirm the booking, and if it corresponds, the system will check the user into their table so that no one will be able to use the table while the user is checked in to the table.
